# Crash-Beat
This is a 1-bit rhythm slasher with several challenging levels!
All game music (excluding the main menu theme) and art were made during Penguin's Jam.

## Controls:
A – Attack Left
D – Attack Right
S – Use Soulfire to boost your combo and heal
